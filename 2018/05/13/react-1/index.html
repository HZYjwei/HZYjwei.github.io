生命周期

  • componentWillMount ： 当前组件已经生成了this实例，将要被渲染，但是这个时候没有生成dom。

  • componentDIdMount：第一次渲染，当前组件的内容已经被渲染，通过ajax获取数据再this.setState({key:vakue})来更改state。

  • componentWillReceiveProps：接受到一个新的props，但是第一次render不会触发。

  • shouldComponentUpdate：boolean，经过diff算法是否需要更新dom

  • componentWillUpdate：将要修改
  • componentDidUpdate：修改后
  • componentWillUnmount：移除组件的事件

路由

Route由上到下匹配。
Route中的exact是严格匹配路由的意思（以下以baidu.com为例）

这样只有输入baidu.com，才能进入Home组件。
但是路径为baidu.com/Com1/Com1_1的时候，由于优先匹配到了/Com1,切不严格匹配，所以直接返回Com_1组件。
NotFound:404页面，没有path，上面所有的路由都没有匹配到。

BrowserRouter和HashRouter

React路由不刷新页面，直接跳转，可以做单页面应用

  • BrowserRouter
    方式：baidu.com/Com
  • HashRouter
    方式：baidu.com/#/Com,以Hash值的方式插入
